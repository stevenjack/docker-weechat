# docker-weechat

Create a Docker container that runs weechat. 


## Setup

To setup, clone this repo and run: 
```sh
docker build docker-weechat
```


## Usage

To run: `docker run -i -t -v ~/.pki:/home/sthulb/.pki <image id>`
